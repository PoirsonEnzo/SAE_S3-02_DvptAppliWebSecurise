Cette version du projet fonctionne via Docker Desktop.
Pour l’essayer, il faut :
1) cloner le projet sur votre PC
2) dans le terminal, dans le dossier NetVOD, taper :
docker compose up -d
3) s’assurer que seul le projet NetVOD est lancé puis taper l’url localhost:8080 sur le navigateur de votre choix.
4) initialiser la base de donner via le bouton « Initialiser la BD »
Note : la base de données est disponible à l’url localhost :8082
